```bash
docker pull ghcr.io/YOUR_USERNAME/mcp-awesome-react-native:latest
```

### Authentication (if private)
```bash
echo $GITHUB_TOKEN | docker login ghcr.io -u YOUR_USERNAME --password-stdin
```

```bash
docker build -t mcp-awesome-react-native:local .
```

### Run Locally
```bash
docker run -p 3000:3000 mcp-awesome-react-native:local
```

### Using Docker Compose
